# Vibe Controller

![Demo](demo.gif)

Xbox コントローラーで macOS を操作するシステムレベルのツール。

**[English](README.md)** | **[简体中文](README_zh-CN.md)** | **[繁體中文](README_zh-TW.md)**

## 機能

- **メニューバー常駐** - ウィンドウを閉じてもバックグラウンドで動作
- **ビジュアルボタンマップ** - 現在のボタンマッピングを直感的に表示
- **バックグラウンド動作** - IOKit HID ベース、ウィンドウがフォーカスを失っても操作可能
- **App Switcher** - Back ボタン長押しで Cmd+Tab のようにアプリを切り替え
- **プロファイルホイール** - L3 を押してプロファイルを素早く切り替え
- **プロファイル自動切替** - アプリが切り替わると自動的にプロファイルを切り替え
- **App Exposé モード** - D-Pad でウィンドウを操作
- **カスタマイズ可能** - 任意のボタンを任意のアクションに設定可能

## クイックスタート

### 方法1：リリース版をダウンロード

[GitHub Releases](https://github.com/bobmayuze/vibeController/releases/tag/alpha) から最新版をダウンロード

### 方法2：Xcode で実行

1. `VibeController.xcodeproj` を開く
2. Cmd+R で実行
3. 初回起動時にアクセシビリティ権限を許可

## デフォルトボタンマッピング

| コントローラー | 機能 |
|--------------|------|
| **左スティック** | マウス移動 |
| **右スティック** | スクロール |
| **A** | 左クリック |
| **B** | 右クリック |
| **X** | コピー (⌘C) |
| **Y** | ペースト (⌘V) |
| **LB** | 元に戻す (⌘Z) / App Switcher で前のアプリ |
| **RB** | Option+Space / App Switcher で次のアプリ |
| **LT** | ドラッグモード（押しながらファイル/テキストをドラッグ） |
| **RT** | Enter |
| **L3** | プロファイルホイール |
| **R3** | Esc |
| **Start** | コマンドパレット (⌘⇧P) |
| **Back** | App Switcher (⌘Tab) |
| **D-Pad 上/下** | 矢印キー |
| **D-Pad 左/右** | Option + 矢印キー（単語移動） |

### コンボキー

| コンボ | 機能 |
|-------|------|
| **LT + D-Pad** | Shift + 矢印キー（テキスト選択） |

## App Switcher の使い方

1. **Back を長押し** → アプリスイッチャーを開く
2. **Back + RB** → 次のアプリ
3. **Back + LB** → 前のアプリ
4. **Back を離す** → 選択を確定

## プロファイル自動切替

アクティブなアプリに基づいてプロファイルを自動切替：

1. 設定で「プロファイル自動切替」を有効化
2. プロファイルの「関連付けアプリ管理」をクリック
3. そのプロファイルに関連付けるアプリを選択
4. 関連付けのないアプリ用にデフォルトプロファイルを設定

関連付けられたアプリに切り替えると、プロファイルが自動で変わり通知が表示されます。

## 権限

初回起動時にアクセシビリティ権限が必要です：

**システム設定 → プライバシーとセキュリティ → アクセシビリティ → Vibe Controller を許可**

## 技術スタック

- **Swift + SwiftUI** - ネイティブ macOS アプリ
- **IOKit HID** - コントローラー入力を直接読み取り、バックグラウンド動作をサポート
- **CoreGraphics** - マウスとキーボードのシミュレーション
- **MenuBarExtra** - メニューバー統合
