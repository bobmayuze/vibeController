# Changelog

## [Unreleased]

### Added
- **Auto Profile Switch**: Automatically switch profiles based on active app
  - Associate apps with specific profiles
  - Set a default profile for unassociated apps
  - Profile switch notification overlay
- **App Exposé Mode**: Navigate windows using D-Pad
  - D-Pad for window selection
  - A to confirm, B to cancel
- **Modifier-Only Chords**: Support chord mappings with only modifier keys (no main button required)
- Active app monitoring via `NSWorkspace` notifications

### Improved
- Enhanced chord mapping system with better modifier key handling
- UI localization for new features (English, Japanese, Simplified Chinese, Traditional Chinese)
